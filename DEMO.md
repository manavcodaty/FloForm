# Klerki Demo Script

## Persona
Student with ADHD applying for a scholarship.

## Demo flow
1. Sign in with Clerk and click **Start new application**.
2. Upload a resume PDF and optionally an ID image, then paste any missing text in the fallback area.
3. Click **Extract my profile** and narrate live streaming updates from the run screen.
4. Open **Profile Review** and show extracted values with confidence badges and evidence snippets.
5. Open **Mapping Review** and point out two low-confidence fields pinned as **Needs approval**.
6. Approve flagged fields and edit one value manually.
7. Open the hosted scholarship form and show highlighted auto-filled fields.
8. Request final approval, approve it, then click **Submit application**.
9. Open the submitted summary page and export JSON.
